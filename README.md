# projeto-criptografia
Projeto com fins acadêmicos com o objetivo de:

Desenvolver uma aplicação com as seguintes funções: Criptograr/Decriptografar um arquivo com criptografia simétrica, criptografar/decriptografar com chave assimetrica, proceder esteganografia de informação. 

Descrição detalhada do funcionamento da aplicação, sendo que deve ter histórico dos eventos realizados, informando horário do usoa, arquivos utilizados, função utilizada, tempo que levou para executar cada atividade (processamento), permitindo consultar essas informaçóes em um arquivo.
